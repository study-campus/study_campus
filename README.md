# 🎓 Study Campus (스터디 캠퍼스)

나의 완벽한 학습 파트너, **Study Campus** 프로젝트입니다. 
웹(Web), 프로그레시브 웹 앱(PWA), 그리고 크로스 플랫폼 모바일 앱(React Native)을 완벽하게 지원하도록 설계된 하이브리드 아키텍처를 채택하고 있습니다.

## 🌟 프로젝트 개요
* **목표:** 기존 메가스터디급의 체계적인 강좌 인프라를 PWA 및 모바일 앱 환경에서도 깨짐 없이 쾌적하게 제공합니다.
* **주요 특징:** * 📱 **반응형 웹 및 PWA:** 모바일 크롬/사파리에서 앱처럼 설치 가능 (`manifest.json`, `sw.js` 적용)
  * 🛡️ **UI/UX 충돌 방지:** Z-index 계층화 및 Overlay(방어막) 패턴을 적용하여 터치 씹힘 및 겹침 현상 원천 차단
  * ⚡ **성능 최적화:** `defer` 스크립트 로드 및 이벤트 위임(Event Delegation)을 통한 렌더링 지연 해결
  * ⚛️ **모바일 앱 확장성:** React Native WebView를 이용한 빠르고 안정적인 네이티브 앱(iOS/Android) 래핑 지원

---

## 🗂️ 프로젝트 아키텍처 (Architecture)

각 기술 스택이 충돌하지 않도록 프론트엔드, 백엔드, 모바일 앱 영역을 철저히 분리하여 관리합니다.
